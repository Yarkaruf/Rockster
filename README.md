###  PROFILE

   * __`ARTS_item`__ |& __`PROJECTS_item`__
        * `PROJECTS_item`
            Картинка, название, тэги (языки)
        * `ARTS_item`
            Картинка, название, тэги
   * __About__
        * Информация о пользователе: описание, ссылки
   * __Likes__
        * Список работ, которые лайкнул пользователь
   * __Following__
        * Список подписок на профили пользователей
   * __Followers__
        * Списон подписчиков профиля
   * __Trophies__
        * EVENTS Trophies 
                * картинка ивента, дата проведения, место
		        * Победитель, призер `events_tags`
        * ?ANY Trophies?
		        * Различные дипломы, сертификаты, заслуги
